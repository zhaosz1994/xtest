## 系统简介

### 什么是 xTest？

xTest 是一个专业的测试管理平台，为开发团队和测试团队提供完整的测试生命周期管理解决方案。系统支持测试计划制定、测试用例管理、测试执行跟踪、测试报告生成等核心功能，帮助团队提升测试效率和质量。

### 核心特性

- 📋 **测试计划管理** - 创建、执行、跟踪测试计划
- 🧪 **测试用例库** - 多层级用例管理，支持多种测试类型
- 📊 **测试报告** - 自动生成测试报告，支持多种格式导出
- 👥 **团队协作** - 用户权限管理，实时协作
- 🔔 **实时通知** - WebSocket 实时推送消息
- 📈 **数据可视化** - 测试进度、通过率等数据可视化展示
- 💬 **社区功能** - 技术讨论、经验分享、知识沉淀

### 用户角色

系统支持以下用户角色：

| 角色 | 权限说明 |
|------|---------|
| **管理员** | 拥有系统所有权限，包括用户管理、系统配置等 |
| **测试经理** | 可创建测试计划、管理测试用例、生成测试报告 |
| **测试工程师** | 可执行测试、更新测试状态、提交测试结果 |
| **开发人员** | 可查看测试结果、参与社区讨论 |

## 快速入门

### 登录系统

1. 打开浏览器，访问系统地址（如：`http://localhost:3000`）
2. 在登录页面输入用户名和密码
3. 点击"登录"按钮

### 注册新账户

1. 在登录页面点击"立即注册"链接
2. 填写注册信息：
   - 用户名（必填）
   - 邮箱（必填）
   - 密码（至少6位）
3. 点击"注册"按钮
4. 注册成功后，使用新账户登录

### 界面概览

登录后，您将看到系统主界面，包含以下部分：

- **顶部导航栏**：系统Logo、全局搜索、通知中心、用户信息
- **左侧菜单栏**：功能模块导航
- **主内容区**：当前模块的操作界面
- **右侧面板**：快捷操作、实时数据

## 主要功能模块

测试概览页面展示系统的整体测试情况，帮助您快速了解测试进度和质量。

### 测试计划管理

测试计划是测试工作的核心，用于规划和管理测试活动。

#### 创建测试计划

1. 点击左侧菜单"测试计划"
2. 点击页面右上角的"+"按钮
3. 选择"新建测试计划"
4. 填写计划信息：
   - **计划名称**（必填）：清晰描述测试计划
   - **负责人**：选择计划负责人
   - **状态**：未开始/进行中/已完成/阻塞
   - **测试阶段**：冒烟测试/功能测试/回归测试等
   - **计划开始时间**和**结束时间**
   - **关联项目**：选择关联的项目
   - **描述**：详细说明测试计划内容
5. 点击"创建"按钮

#### 筛选测试计划

使用页面顶部的筛选标签快速定位计划：

- **我负责的**：显示您负责的测试计划
- **未完成的**：显示未完成的测试计划
- **已完成的**：显示已完成的测试计划

#### 编辑测试计划

1. 在测试计划列表中，点击计划名称
2. 进入计划详情页面
3. 点击"编辑"按钮
4. 修改计划信息
5. 点击"保存"按钮

#### 删除测试计划

1. 在测试计划列表中，点击计划右侧的"删除"按钮
2. 确认删除操作
3. **注意**：删除后可在回收站恢复

### 用例库管理

用例库是测试用例的集中管理中心，支持多层级结构和多种测试类型。

#### 用例库结构

用例库采用四级结构：

```
用例库
  └─ 模块
      └─ 一级测试点
          └─ 二级测试点（测试用例）
```

#### 创建测试用例

**单个创建**：

1. 点击左侧菜单"用例库"
2. 选择目标模块和测试点
3. 点击右上角"+"按钮，选择"新建用例"
4. 填写用例信息：
   - **用例名称**（必填）
   - **优先级**：P0/P1/P2/P3
   - **测试类型**：功能测试/性能测试/安全测试等
   - **负责人**：选择用例负责人
   - **测试阶段**：选择测试阶段
   - **测试环境**：选择测试环境
   - **前置条件**：执行测试前需满足的条件
   - **测试步骤**：详细的测试操作步骤
   - **预期结果**：期望的测试结果
5. 点击"创建"按钮

**批量创建**（推荐）：

1. 在用例库页面，点击"批量创建"按钮
2. 进入批量创建界面
3. 填写表格（支持Excel复制粘贴）：
   - 用例名称（必填）
   - 优先级
   - 测试类型
   - 负责人
   - 测试阶段
   - 测试环境
4. 点击"添加一行"可增加更多用例
5. 点击"批量默认值"可快速设置默认值
6. 点击"批量保存"完成创建

#### 筛选测试用例

使用筛选功能快速定位用例：

- **按优先级筛选**：P0/P1/P2/P3
- **按状态筛选**：未执行/通过/失败/阻塞
- **按类型筛选**：功能测试/性能测试等
- **按负责人筛选**：选择特定负责人

#### 导入/导出用例

**导入用例**：

1. 点击"导入"按钮
2. 下载导入模板
3. 按模板格式填写用例数据
4. 上传Excel文件
5. 系统自动解析并导入

**导出用例**：

1. 选择要导出的用例（可多选）
2. 点击"导出"按钮
3. 选择导出格式（Excel/CSV）
4. 下载导出文件

#### 执行测试用例

1. 在测试计划中，找到要执行的用例
2. 点击"执行"按钮
3. 填写执行结果：
   - **执行状态**：通过/失败/阻塞
   - **实际结果**：记录实际测试结果
   - **缺陷链接**：关联的缺陷链接（可选）
   - **备注**：其他说明信息
4. 点击"提交"按钮

### 测试报告管理

测试报告用于汇总测试结果，展示测试质量和进度。

#### 创建测试报告

1. 点击左侧菜单"测试报告"
2. 点击右上角"+"按钮，选择"新建测试报告"
3. 填写报告信息：
   - **报告名称**（必填）
   - **报告类型**：日报/周报/最终报告
   - **创建人**：自动填充当前用户
   - **关联项目**：选择关联项目
   - **关联测试计划**：选择关联的测试计划
   - **报告周期**：选择报告时间范围
   - **测试总结**：整体测试情况说明
4. 点击"创建"按钮

#### 查看测试报告

1. 在测试报告列表中，点击报告名称
2. 查看报告详情：
   - 测试概览数据
   - 测试执行统计
   - 缺陷统计
   - 风险分析
   - 测试结论

#### 导出测试报告

1. 打开测试报告详情页
2. 点击"导出"按钮
3. 选择导出格式：
   - **Markdown**：适合在线查看和编辑
   - **PDF**：适合打印和归档
4. 下载导出文件

2. 选择"用户管理"标签
3. 点击"添加用户"按钮
4. 填写用户信息：

   - 电话

**编辑用户**：

1. 在用户列表中，点击用户右侧的"编辑"按钮
2. 修改用户信息
3. 点击"保存"按钮

**删除用户**：

1. 在用户列表中，点击用户右侧的"删除"按钮
2. 确认删除操作
3. **注意**：删除用户后，该用户将无法登录系统

**添加项目**：

1. 在配置中心，选择"项目配置"标签
2. 点击"添加项目"按钮
3. 填写项目信息：
   - 项目名称（必填）

   - 项目负责人

#### 环境配置

**添加测试环境**：

1. 在配置中心，选择"环境配置"标签
2. 点击"添加环境"按钮
3. 填写环境信息：
   - 环境名称（必填）：如"开发环境"、"测试环境"
   - 环境地址
   - 环境描述
   - 环境状态：可用/维护中/已下线
4. 点击"保存"按钮

### 社区功能

社区功能为用户提供技术交流和经验分享的平台。

#### 访问社区

1. 点击顶部导航栏的"社区"链接
2. 进入社区首页

#### 发布帖子

1. 点击社区页面右上角的"发布帖子"按钮
2. 填写帖子内容：
   - **标题**（必填）：简洁明了的标题

- **全部帖子**：显示所有帖子

- **排序方式**：
  - 按时间：最新发布的帖子在前

1. 在社区页面顶部，找到搜索框
2. 输入关键词
3. 点击"搜索"按钮

**发表评论**：

1. 打开帖子详情页
2. 在评论框中输入评论内容
3. 支持@提及其他用户
4. 点击"发表评论"按钮

**点赞帖子**：

2. 点赞数会实时更新

点击右上角用户名，进入个人中心：

- **我的帖子**：查看和管理自己发布的帖子
- **我的评论**：查看自己的评论历史

## 常见问题

### 1. 忘记密码怎么办？

**解决方法**：
- 联系系统管理员重置密码
- 管理员可在"用户管理"中重置用户密码

### 2. 测试计划无法创建？

**可能原因**：
- 表单数据未填写完整
- 网络连接异常
- 后端服务未启动

**解决方法**：
- 检查必填项是否都已填写
- 检查网络连接
- 刷新页面重试

### 3. 测试用例执行后状态未更新？

**可能原因**：
- 网络延迟
- 浏览器缓存问题

**解决方法**：
- 刷新页面
- 清除浏览器缓存
- 检查网络连接

### 4. 无法上传附件？

**可能原因**：
- 文件大小超过限制
- 文件格式不支持
- 网络连接异常

**解决方法**：
- 检查文件大小（建议不超过10MB）
- 确认文件格式支持（支持常见图片、文档格式）
- 检查网络连接

### 5. 实时通知不工作？

**可能原因**：
- WebSocket服务未启动
- 浏览器不支持WebSocket
- 网络防火墙限制

**解决方法**：
- 检查网络连接
- 更换浏览器（推荐Chrome、Firefox）
- 联系系统管理员

### 6. 数据导出失败？

**可能原因**：
- 数据量过大
- 网络超时

**解决方法**：
- 减少导出数据量
- 分批次导出
- 检查网络连接
