## 五、性能审查

### 5.1 数据库查询优化

#### ⚠️ 当前问题
1. **循环查询**：在for循环中执行SQL
2. **N+1问题**：每个用例单独查询

#### 💡 优化建议
```javascript
// 优化前：循环查询
for (const caseItem of cases) {
    const [reviewerRows] = await connection.execute(
        `SELECT id, status FROM case_reviewers WHERE case_id = ? AND reviewer_id = ?`,
        [caseItem.id, currentUser.id]
    );
}

// 优化后：批量查询
const caseIds = cases.map(c => c.id);
const placeholders = caseIds.map(() => '?').join(',');
const [allReviewerRows] = await connection.execute(
    `SELECT case_id, id, status FROM case_reviewers 
     WHERE case_id IN (${placeholders}) AND reviewer_id = ?`,
    [...caseIds, currentUser.id]
);
const reviewerMap = new Map(allReviewerRows.map(r => [r.case_id, r]));
```
